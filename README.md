# Tobacco-Use-Mortality-
Project Overview
This project analyzes the relationship between tobacco use and mortality rates using publicly available health datasets. The goal is to uncover patterns, quantify risks, and present clear insights into how tobacco consumption contributes to preventable deaths worldwide.
Key Features
- Data Cleaning & Preparation: Structured raw health data for accurate analysis.
- Exploratory Analysis: Identified trends in tobacco use across age groups, genders, and regions.
- Statistical Modeling: Applied hypothesis testing
